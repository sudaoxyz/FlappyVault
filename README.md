# EvalSocial 🔒
EvalSocial is a web3 automation helper.

## 安装 💽

### 在 Chrome 中加载扩展

1. 克隆此仓库

2. 安装依赖
   ```bash
   npm install
   ```
   
3. 构建：
   ```bash
   npm run build
   ```
   
4. 打开 Chrome，然后转到 chrome://extensions/

5. 在页面右上角启用 "开发者模式"。

6. 点击 "加载已解压的扩展程序"，并选择您刚刚构建生成的 EvalSocial > dist 目录。
    

EvalSocial 扩展现在应该可以在 Chrome 中使用了！
